Add Eligibility Period:

Add Eligibiles:

Please select an eligibility period that the eligibles are a in. The select a file to upload eligible information into the system. The file must be a csv file, where the columns are: eligible name, eligible kerberos. After choosing a file, hit Upload to load the eligibles from the csv file into the system to the set eligibility period.

Export Eligibile Information:

Select the eligibility period for which you would like to export all eligible information. Once you hit Export, eligible information will be downloaded in the form of a csv file, where the columns are: name, kerberos, [requirement, complete, comments]+.

Edit Eligibile Information:

Click here to edit a specific eligible's information.
